# 🔥 Hot Reload в Flutter

Flutter поддерживает Hot Reload (как в React Native), что позволяет быстро обновлять UI без полной перезагрузки приложения.

## 🚀 Как использовать:

### После запуска `flutter run`:

1. **Hot Reload (r)** - Быстрое обновление кода
   - Нажмите `r` в терминале где запущено `flutter run`
   - Обновляет UI за секунды
   - Сохраняет состояние приложения

2. **Hot Restart (R)** - Полный перезапуск с сохранением состояния
   - Нажмите `R` (заглавная) в терминале
   - Используйте, когда Hot Reload не помог
   - Перезапускает виджеты, но сохраняет некоторые данные

3. **Full Restart (q + запуск заново)** - Полный перезапуск
   - Нажмите `q` для остановки
   - Запустите `flutter run` заново
   - Используйте при серьезных изменениях

## 💡 Когда использовать:

- **Hot Reload (r)**: Изменения в UI, стили, текст
- **Hot Restart (R)**: Изменения в логике, новые виджеты, изменения в initState
- **Full Restart**: Изменения в native коде, новые зависимости

## 📱 Для текущего обновления:

1. Убедитесь, что приложение запущено (`flutter run`)
2. Нажмите **`R`** (заглавная) для Hot Restart
3. Кнопка помощи должна появиться в правом верхнем углу

## 🔧 Альтернатива:

Если Hot Reload не работает, остановите приложение (`q`) и запустите заново:
```bash
./scripts/run_emulator_with_logs.sh
```
