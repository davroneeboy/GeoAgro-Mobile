# Обновление 3.2.0

## Основные изменения

### 1. Исправление отображения статуса плантаций
- Плантации с `is_rejected: true` теперь корректно отображаются как "Rad etilgan" вместо "Ko'rib chiqilmoqda"
- Статус отображается цветным индикатором (кружок) вместо текстового бейджа:
  - 🟢 Зеленый - "Tasdiqlangan"
  - 🔴 Красный - "Rad etilgan"
  - 🟠 Оранжевый - "Ko'rib chiqilmoqda"

### 2. Блокировка редактирования и удаления подтвержденных плантаций
- Плантации со статусом `is_checked: true` нельзя редактировать и удалять
- Кнопки редактирования и удаления скрыты для подтвержденных плантаций
- При попытке удалить показывается сообщение: "Tasdiqlangan plantatsiyani o'chirib bo'lmaydi."

### 3. Упрощение удаления фотографий
- Убрана кнопка с крестиком для удаления фотографий
- Удаление доступно через bottom sheet при нажатии на фотографию:
  - "O'chirish" (Удалить)
  - "Galereyadan tanlash" (Выбрать из галереи) - только для специальных пользователей
  - "Kameradan olish" (Сфотографировать)

### 4. Ограничение доступа к галерее
- Обычные пользователи (`is_specialuser == false`) не могут выбирать фотографии из галереи, только фотографировать камерой
- Специальные пользователи (`is_specialuser == true`) могут выбирать из галереи и фотографировать камерой

### 5. Проверка разницы территории
- При создании/редактировании плантации проверяется разница между площадью полигона и общей площадью
- Если разница превышает 15%, добавление/сохранение блокируется с сообщением об ошибке

### 6. Исправление отправки user_location
- `user_location` отправляется как объект (не массив), как в Postman
- Формат: `{"latitude": 41.215, "longitude": 69.341}`

### 7. Отображение территории тумана на карте
- На карте отображается граница тумана (district) пользователя
- Отображается только территория тумана пользователя, а не всего региона

### 8. Улучшена страница профиля
- На странице профиля теперь отображается список разрешений (доступов), которые требуются приложению
- Пользователь может видеть, какие разрешения запрашивает приложение
