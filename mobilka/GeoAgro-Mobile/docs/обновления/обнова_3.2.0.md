### 4. Улучшение обработки ошибок при удалении плантаций

#### Проблема
При удалении плантации сервер возвращал ошибку в невалидном JSON формате: `{error: No Plantation matches the given query.}`, что вызывало ошибки парсинга.

#### Решение
Улучшен парсинг ошибок с поддержкой нескольких вариантов извлечения сообщения:
- `{error: message}` - с фигурными скобками
- `error: message` - без фигурных скобок
- Поиск текста после "error:" до конца строки или до закрывающей скобки

#### Файлы
- `lib/src/feature/home/vm/home_page_vm.dart`

### 5. Исправление ошибки "used after being disposed" в FarmersStatisticsVm

#### Проблема
При первом входе с новым токеном страница "Fermerlar" не загружалась из-за ошибки "A FarmersStatisticsVm was used after being disposed".

#### Решение
- Добавлен флаг `_isDisposed` для отслеживания состояния ViewModel
- Создан метод `_safeNotifyListeners()` для безопасного вызова `notifyListeners()`
- Добавлены проверки на `_isDisposed` во всех асинхронных методах:
  - `initialize()`
  - `fetchStatistics()`
  - `searchByInn()`
  - `clearSearch()`

### 8. Ограничение доступа к галерее на основе статуса пользователя

#### Изменение
Добавлена проверка `is_specialuser` из API `/api/user_info/`. Обычные пользователи не могут выбирать фотографии из галереи, только фотографировать камерой.

#### Реализация

**ViewModel уровень:**
- Добавлен геттер `isSpecialUser` в `DetailVM` и `EditVM`
- Метод `loadUserInfo()` загружает значение `is_specialuser` из `AppStorage`
- Метод `showImagePicker()` скрывает опцию "Gallery" для обычных пользователей

**UI уровень:**
- В `ImageUploadListWidget` добавлен параметр `isSpecialUser`
- Опция "Galereyadan tanlash" в bottom sheet скрывается, если `isSpecialUser == false`
- При инициализации страниц создания и редактирования вызывается `loadUserInfo()`

**Логика:**
- Если `is_specialuser == true`: пользователь может выбирать фото из галереи и фотографировать камерой
- Если `is_specialuser == false`: пользователь может только фотографировать камерой, опция галереи скрыта

#### Файлы
- `lib/src/feature/detail_page/vm/detail_vm.dart`
- `lib/src/feature/edit/vm/edit_vm.dart`
- `lib/src/feature/detail_page/view/widgets/images_upload_widget.dart`
- `lib/src/feature/detail_page/view/pages/detail_page.dart`

### Защита от использования disposed ViewModel

Паттерн защиты:
```dart
bool _isDisposed = false;

void _safeNotifyListeners() {
  if (!_isDisposed) {
    notifyListeners();
  }
}

@override
void dispose() {
  _isDisposed = true;
  // ... остальной код dispose
  super.dispose();
}
```

### Ограничение доступа к галерее

Проверка статуса пользователя:
```dart
// В ViewModel
bool _isSpecialUser = false;
bool get isSpecialUser => _isSpecialUser;

Future<void> loadUserInfo() async {
  _isSpecialUser = await AppStorage.$readBool(key: StorageKey.isSpecialUser) ?? false;
}

// В showImagePicker
if (_isSpecialUser) ...[
  const Divider(),
  ListTile(
    leading: const Icon(Icons.photo_library),
    title: const Text('Gallery'),
    onTap: () => Navigator.pop(context, ImageSource.gallery),
  ),
],
```

В виджете загрузки изображений:
```dart

  ListTile(
    leading: const Icon(Icons.photo_library),
```
