# Обновление 3.3.2

## Основные изменения

### 1. Исправлен маппинг типов субсидий (direction)
- **Проблема:** Порядок типов субсидий в мобильном приложении не совпадал с бэкендом. Например, `direction: 1` отображался как "Limon" вместо "Tomchilatib sug'orish tizimlari uchun".
- **Решение:** Маппинг `subsidyTypes` приведён в соответствие с бэкендом:
  - 1 → Tomchilatib sug'orish tizimlari uchun
  - 2 → Quduq/nasos stantsiyalari uchun
  - 3 → Ko'chat uchun
  - 4 → Shpalier uchun
  - 5 → Limon uchun
  - 6 → Muqobil energiya uchun
- Строковые константы обновлены на полные названия.
- **Файл:** `lib/localization/app_strings.dart`

### 2. Добавлена загрузка существующих субсидий при редактировании плантации
- **Проблема:** При открытии страницы редактирования плантации существующие субсидии не загружались в форму. Переключатель "Subsidiya" оставался выключенным, а список субсидий — пустым. При сохранении субсидии затирались.
- **Решение:** В `getPlantationDetail()` добавлена загрузка `plantationModel.subsidies` в `selectedEditSubsidy`, а также автоматическое включение переключателей `switchSubsidiya` и `switchEfficiency` при наличии данных.
- **Файл:** `lib/src/feature/edit/vm/edit_vm.dart`

### 3. Добавлено отображение направления субсидии на странице просмотра
- **Проблема:** На странице просмотра плантации в блоке "Subsidiyalar" не отображалось поле "Yo'nalish" (направление субсидии).
- **Решение:** Добавлено поле "Yo'nalish" с иконкой категории, отображающее название типа субсидии по маппингу `subsidyType`.
- **Файл:** `lib/src/feature/home/view/pages/plantation_view_page.dart`

### 4. Выбор года субсидии через колёсико (CupertinoPicker)
- **Было:** На странице создания — выбор даты через `showDatePicker` (полный календарь), на странице редактирования — ручной ввод числа.
- **Стало:** На обеих страницах — выбор года через прокрутку колёсика (`CupertinoPicker`) в модальном окне, диапазон: 2000 — текущий год.
- **Файлы:**
  - `lib/src/core/widgets/year_wheel_picker.dart` (новый виджет)
  - `lib/src/feature/detail_page/view/widgets/add_subsidiya_bottom_shit.dart`
  - `lib/src/feature/edit/view/widget/edit_subsidy_bottom_shit.dart`

### 5. Маска форматирования суммы субсидии
- **Было:** Поле суммы принимало только цифры без форматирования — при вводе больших сумм сложно определить порядок числа.
- **Стало:** Применён `ThousandsSeparatorInputFormatter` с разделителем тысяч (пробел): `1 500 000`. Добавлен суффикс "so'm" справа в поле ввода. Пробелы корректно убираются при парсинге перед отправкой.
- **Файлы:**
  - `lib/src/feature/detail_page/view/widgets/add_subsidiya_bottom_shit.dart`
  - `lib/src/feature/edit/view/widget/edit_subsidy_bottom_shit.dart`
  - `lib/src/feature/detail_page/view/widgets/detail_text_fild_widget.dart` (добавлен параметр `suffixText`)
  - `lib/src/feature/detail_page/vm/detail_vm.dart` (парсинг суммы)
  - `lib/src/feature/edit/vm/edit_vm.dart` (парсинг суммы)

### 6. Проверка на дубликат номера договора субсидии
- **Было:** Можно было добавить несколько субсидий с одинаковым номером договора.
- **Стало:** При попытке добавить субсидию с уже существующим номером договора показывается алерт: "Bu shartnoma raqami allaqachon qo'shilgan. Shartnoma raqami bir xil bo'lishi mumkin emas".
- **Файлы:**
  - `lib/src/feature/detail_page/view/widgets/add_subsidiya_bottom_shit.dart`
  - `lib/src/feature/edit/view/widget/edit_subsidy_bottom_shit.dart`

### 7. Исправлены Riverpod-антипаттерны и утечки памяти
- Заменён `ref.watch()` на `ref.read()` в callback-методах `addSubsidiyaList` обоих VM (нельзя использовать `ref.watch` вне `build`).
- Исправлен вызов `notifyListeners()` извне `ChangeNotifier` в `edit_subsidy_bottom_shit.dart` — заменён на `vm.setSubsidiya()`.
- Добавлен `notifyListeners()` в `resetSubsudy()` в `edit_vm.dart` для корректного обновления UI.
- `FixedExtentScrollController` в `YearWheelPicker` теперь корректно утилизируется через `.whenComplete(() => scrollController.dispose())`.
- **Файлы:**
  - `lib/src/feature/detail_page/vm/detail_vm.dart`
  - `lib/src/feature/edit/vm/edit_vm.dart`
  - `lib/src/feature/edit/view/widget/edit_subsidy_bottom_shit.dart`
  - `lib/src/core/widgets/year_wheel_picker.dart`

### 8. Защита от overflow при открытии клавиатуры в bottom sheet
- Оба bottom sheet (создание и редактирование субсидий) обёрнуты в `SingleChildScrollView` с учётом `MediaQuery.viewInsets.bottom` — при открытии клавиатуры контент прокручивается и не обрезается.
- **Файлы:**
  - `lib/src/feature/detail_page/view/widgets/add_subsidiya_bottom_shit.dart`
  - `lib/src/feature/edit/view/widget/edit_subsidy_bottom_shit.dart`

### 9. Обновлены Gradle и Android Gradle Plugin
- AGP обновлён с 8.6.0 до 8.9.1 для совместимости с `androidx.core:core-ktx:1.17.0`.
- Gradle обновлён с 8.7 до 8.11.1 (минимальная версия для AGP 8.9.1).
- **Файлы:**
  - `android/settings.gradle`
  - `android/gradle/wrapper/gradle-wrapper.properties`

### 10. Устранены предупреждения анализатора
- Удалены неиспользуемые переменные `theme` в методах `_buildModerationCommentCard` и `_buildCommentCard`.
- Убран ненужный оператор `!` на `summaryCard`, который уже проверен на `null`.
- **Файл:** `lib/src/feature/home/view/pages/plantation_view_page.dart`
