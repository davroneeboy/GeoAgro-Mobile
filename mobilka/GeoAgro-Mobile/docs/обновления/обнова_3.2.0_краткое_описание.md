# Обновление 3.2.0 - Краткое описание

## Что нового:

✅ **Исправлено отображение статусов плантаций** - теперь статусы отображаются цветными индикаторами (🟢 подтверждено, 🔴 отклонено, 🟠 на проверке)

✅ **Защита подтвержденных плантаций** - подтвержденные плантации нельзя редактировать или удалять

✅ **Улучшен интерфейс работы с фотографиями** - удаление фотографий через bottom sheet, убрана кнопка с крестиком

✅ **Ограничение доступа к галерее** - обычные пользователи могут только фотографировать камерой, специальные пользователи - также выбирать из галереи

✅ **Проверка разницы территории** - при создании/редактировании плантации проверяется, что разница между площадью полигона и общей площадью не превышает 15%

✅ **Исправлена отправка координат** - `user_location` теперь корректно отправляется на сервер

✅ **Добавлено отображение территории тумана** - на карте отображается граница тумана пользователя

✅ **Улучшена страница профиля** - теперь видно какие разрешения (доступы) требуются приложению
