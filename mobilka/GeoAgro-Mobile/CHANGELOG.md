# Changelog

## [3.2.0] - 2024

### ✨ Добавлено

- **Клиентская валидация на странице создания фермера**
  - Валидация в реальном времени для всех полей формы
  - Отображение ошибок под каждым полем
  - Обязательные поля отмечены звездочкой (*)
  - Ограничение ввода: поля имен (Asoschi, Rahbar) принимают только буквы
  - Защита от XSS инъекций (санитизация всех текстовых полей)

- **Улучшения на странице фермеров**
  - Кнопка "Янги фермер" добавлена в хедер страницы
  - Бесконечная прокрутка (автоматическая загрузка при прокрутке до конца)
  - Встроенный поиск по INN (без отдельной страницы)

- **Отображение границ региона на карте**
  - Автоматическое отображение границ района пользователя
  - Загрузка границ из GeoJSON файлов
  - Белая обводка границ региона

- **Улучшения UI**
  - Унифицирован цвет фона (темно-синий) на всех страницах
  - Заменены PNG изображения пустого состояния на SVG
  - Исправлены переполнения текста (overflow errors)

### 🐛 Исправлено

- Исправлены ошибки переполнения текста ("overflowed by X pixel")
- Улучшена валидация полей формы создания фермера
- Исправлена минимальная длина для адреса и ИНН
- Улучшена обработка ошибок валидации

### 🔧 Технические улучшения

- Добавлен сервис `GeoJsonService` для работы с геоданными
- Оптимизирована загрузка данных на странице фермеров
- Улучшена производительность при работе с большими списками
