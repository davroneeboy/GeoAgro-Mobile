# Исправления для страницы модерации

## Проблемы, которые были исправлены:

### 1. Неправильное вычисление totalPages
**Проблема:** Использовался `moderations.length` как `pageSize`, что приводило к неправильному вычислению общего количества страниц.

**Решение:** Установлен фиксированный `pageSize = 20`.

### 2. Ошибки 404 при переходе на несуществующие страницы
**Проблема:** Пользователь мог перейти на страницу 101, хотя всего страниц 42, что приводило к ошибкам 404.

**Решение:** 
- Добавлена проверка валидности номера страницы
- При получении 404 автоматический возврат на первую страницу
- Добавлена кнопка "Вернуться на первую страницу" при ошибках

### 3. Отсутствие обработки состояний загрузки и ошибок
**Проблема:** Пользователь не видел, когда данные загружаются или произошла ошибка.

**Решение:**
- Добавлено состояние `loading` с индикатором загрузки
- Добавлено состояние `error` с отображением ошибок
- Добавлено сообщение "Нет данных для отображения" при пустом списке

### 4. Улучшенная пагинация
**Проблема:** Кнопки пагинации не проверяли границы страниц.

**Решение:**
- Добавлены проверки для кнопок "Назад" и "Вперед"
- Кнопки отключаются при достижении границ
- Добавлена проверка на превышение общего количества страниц

### 5. Валидация данных от API
**Проблема:** Отсутствовала проверка структуры ответа от API.

**Решение:** Добавлена проверка существования `response.data` и `response.data.results`.

## Изменения в коде:

1. **Исправлено вычисление totalPages:**
```javascript
const pageSize = 20; // фиксированный размер страницы
const totalPages = Math.ceil(count / pageSize);
```

2. **Добавлена валидация номера страницы:**
```javascript
if (pageNumber > 0) {
  setPage(pageNumber);
  localStorage.setItem('moderationPage', pageNumber);
} else {
  setPage(1);
  localStorage.setItem('moderationPage', 1);
  navigate('/moderation?page=1', { replace: true });
}
```

3. **Улучшена обработка ошибок:**
```javascript
if (error.response?.status === 404) {
  console.log('Страница не найдена, возвращаемся на первую страницу');
  setPage(1);
  localStorage.setItem('moderationPage', 1);
  navigate('/moderation?page=1', { replace: true });
}
```

4. **Добавлены состояния загрузки и ошибок:**
```javascript
const [loading, setLoading] = useState(false);
const [error, setError] = useState(null);
```

## Результат:
- Устранены ошибки 404 при пагинации
- Улучшен пользовательский опыт с индикаторами загрузки
- Добавлена корректная обработка ошибок
- Исправлено отображение общего количества страниц
